# Implementation Plan

## Fix 1 — Broker Registration "Field required" Bug

**File:** `frontend/src/app/dashboard/team/page.tsx`

**What to change:**

- Rename o estado inicial: `name` → `nome`
- Atualizar o reset após POST bem-sucedido: `name` → `nome`
- Atualizar o `value` e `onChange` do input de nome
- Corrigir exibição da tabela: `broker.name` → `broker.nome`

**Why:** O schema Pydantic espera o campo `nome` (português). O frontend enviava `name`, causando falha de validação com "Field required". A tabela também referenciava a chave errada da resposta.
