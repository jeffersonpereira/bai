## FASE 1 — CRÍTICOS

### BUG 1 — Inconsistência `status` vs `situacao` no estado local do frontend

**Arquivo:** `frontend/src/app/dashboard/page.tsx`

O body do PATCH envia `status` (correto — backend aceita). Mas ao atualizar o estado local, usa `status` onde deveria usar `situacao` (nome do campo no objeto retornado pelo backend).

**Correção — linha ~113:**
```javascript
// ANTES:
setProperties(prev => prev.map((p: any) => p.id === id ? { ...p, status: newStatus } : p));

// DEPOIS:
setProperties(prev => prev.map((p: any) => p.id === id ? { ...p, situacao: newStatus } : p));
```

### BUG 2 — `logout()` chamado fora de escopo em AuthContext.tsx

**Arquivo:** `frontend/src/app/context/AuthContext.tsx`

`fetchUser` é definido via `useCallback` com `[]` de deps, e chama `logout()` internamente — mas `logout` é definido depois, fora do closure criado. Em runtime, `logout` é `undefined` no momento em que `fetchUser` é instanciado.

**Correção — substituir a chamada `logout()` por lógica inline dentro de `fetchUser`:**
```javascript
// ANTES (dentro de fetchUser, ~linha 61):
if (response.status === 401) {
  logout();
}

// DEPOIS:
if (response.status === 401) {
  localStorage.removeItem("bai_token");
  setToken(null);
  setUser(null);
  router.push("/");
}
```

E adicionar `router` na dependência do `useCallback`:
```javascript
// ANTES:
const fetchUser = useCallback(async (authToken: string) => { ... }, []);

// DEPOIS:
const fetchUser = useCallback(async (authToken: string) => { ... }, [router]);
```

### BUG 8 — `Promise.all` falha silenciosamente se um request falhar

**Arquivo:** `frontend/src/app/documents/page.tsx` linhas ~242-255

**Correção — usar `Promise.allSettled`:**
```javascript
// ANTES:
const [imoveisRes, proprietariosRes] = await Promise.all([...]);

// DEPOIS:
const [imoveisRes, proprietariosRes] = await Promise.allSettled([
  fetch(`${API}/api/v1/imoveis/user/me`, { headers: authHeader }),
  fetch(`${API}/api/v1/proprietarios/`, { headers: authHeader }),
]);
if (imoveisRes.status === "fulfilled" && imoveisRes.value.ok) {
  const d = await imoveisRes.value.json();
  setImoveis(d.items ?? d);
}
if (proprietariosRes.status === "fulfilled" && proprietariosRes.value.ok) {
  const d = await proprietariosRes.value.json();
  setProprietarios(d.items ?? d);
}
```
